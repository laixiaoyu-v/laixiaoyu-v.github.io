前言

什么是hexo?

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。

为什么要搭建Hexo博客呢？

  1. 我的博客最近刚刚搭建起来趁着热度没减退再搭建一个!
  2. Hexo可以部署在GitHub不需要服务器
  3. 因为好玩!!

准备工具

  1. Node.js
  2. Git
  3. GitHub账号
