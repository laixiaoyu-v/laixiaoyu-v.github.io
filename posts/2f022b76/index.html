关于Github Action

GitHub Actions 是由 GitHub 提供的服務，用於在 GitHub 建置、測試、封裝、發佈或部署任何專案。並可藉此建置端對端 (end-to-end) CI/CD 功能，即持續整合/持續交付 (微軟譯為持續傳遞) 服務。

BilibiliTask

哔哩哔哩(B站)自动完成每日任务，投币，点赞，直播签到，自动兑换银瓜子为硬币，自动送出即将过期礼物，漫画App签到。

GitHub：BilibiliTask

CloudMusic-LevelUp

网易云音乐签到刷听歌量,助你快速升到10级！

GitHub：CloudMusic-LevelUp

EUserv_extend

自动续期EUserv免费IPv6 VPS。

GitHub：EUserv_extend

textnowProtect

TextNow保号任务，给指定号码发短信，支持多账号。 

GitHub：textnowProtect
